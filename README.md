# MultiAI Side-by-Side

A clean, powerful browser-based tool for running **ChatGPT**, **Claude**, and **Grok** side-by-side with excellent prompt management and one-click response transfer via clipboard.

## ✨ Features
- **One-click side-by-side launch** — Opens three perfectly positioned chat windows
- **Central Prompt Composer** — Write once, copy to any model with one click
- **Response Capture & Transfer** — Paste responses and move ideas between AIs instantly
- **Smooth clipboard experience** — Toast notifications + reliable copy
- **Fully local & private** — Everything stays in your browser
- **Auto-save** — Your work persists across refreshes

## How to Use
1. Open `index.html` in any modern browser
2. Click the big **Launch Side-by-Side Chats** button
3. Log into ChatGPT, Claude, and Grok in their own windows
4. Use the Prompt Composer and the colored transfer buttons

- Single-file HTML
- Tailwind CSS (via CDN)

- No build step required

Built for serious parallel LLM workflows.
